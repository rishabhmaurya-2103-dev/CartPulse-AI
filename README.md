# 🛒 CartPulse AI – Cart Abandonment Predictor

CartPulse AI is a machine learning-powered web application that predicts whether a user will abandon their shopping cart or complete a purchase. It analyzes user session behavior and provides real-time risk scoring with visual insights.

---

## 🚀 Features

- 🔍 Predict cart abandonment risk (Low / Medium / High)
- 📊 Interactive dashboard with charts (Chart.js)
- ⚡ Real-time ML prediction using Flask
- 🧠 Behavior-based analysis (views, cart activity, pricing)
- 💡 Smart suggestions to improve conversion
- 🎨 Modern glassmorphism UI

---

## 🧠 How It Works

1. User enters session details:
   - Product price
   - Views
   - Cart activity
   - Number of products viewed

2. ML model analyzes behavior patterns

3. System outputs:
   - Risk Level (Low / Medium / High)
   - Probability %
   - Suggested action (discount, reminder, etc.)

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Machine Learning:** Scikit-learn
- **Frontend:** HTML, CSS, JavaScript
- **Visualization:** Chart.js
